Steps to execute the project:
1. create database named recycon.
2. Rename the .env example to .env and change the database name to recycon
3. Add FILESYSTEM_DRIVER=public
4. Run composer install to install vendor
5. Run the php artisan key:generate to generate app_key
6. Run the database migration and seeders.
7. Admin ditentukan dengan cara menambahkan kata 'admin' saat registrasi username
